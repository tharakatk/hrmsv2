## Overview

Marketing website for **lifeHRMS** — a cloud-based HRMS offering payroll, attendance, leave management, performance reviews, recruitment, and employee self-service.

Features real product screenshots showcasing the actual dashboard, analytics, HR metrics, and attendance tracking modules.

## Quick Start

```bash
# Just open directly
open index.html

# Or use a local server
python3 -m http.server 8000
npx serve .
```

## Deployment (GitHub Pages)

1. Push to GitHub
2. Settings → Pages → Source: `main` / `/ (root)`
3. Live at `https://<username>.github.io/<repo>/`
